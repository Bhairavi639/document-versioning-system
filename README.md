# 📄 Document Versioning System

A web application for comparing different versions of PDF, DOCX, and Excel documents.

Built using FastAPI and React, the system detects changes between document versions and generates downloadable comparison reports.

---

## Features

- Compare PDF documents
- Compare DOCX documents
- Compare Excel workbooks
- Automatic sheet matching
- Key column detection
- Highlight added, removed, and modified records
- Download comparison report
